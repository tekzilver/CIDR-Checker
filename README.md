ğŸ“¥ How to Install

Since this is not published on the Chrome Web Store yet, you must install it manually:

    Download this repository (Code -> Download ZIP) or clone it.
    Extract the files to a folder on your computer.
    Open Google Chrome and navigate to chrome://extensions.
    Toggle Developer mode in the top right corner.
    Click the Load unpacked button.
    Select the folder where you saved the files.
    The Tekzilver CIDR-Checker icon will now appear in your toolbar!

    Click the Tekzilver CIDR-Checker icon in your browser toolbar.
